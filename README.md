# IJ 商品化プロセス（ISO Step） | IJ 상품화 프로세스(ISO Step)

## 概要 / 개요
本ページは **IJコンポーネント事業部の「商品化プロセス（ISO Step）」** を、学習・参照用に整理したものです。  
日本語/韓国語の切替、ならびに **「規定 / 営業視点」** のタブ切替により、プロセスの理解と実務の抜け漏れ防止を目的としています。

이 페이지는 **IJ 컴포넌트 사업부 상품화 프로세스(ISO Step)** 를 학습/참고용으로 보기 좋게 정리한 것입니다.  
일본어/한국어 전환 및 **「규정 / 영업 시점」** 탭 전환을 통해 프로세스 이해와 실무 누락 방지를 목표로 합니다.

## 注意 / 주의
- 本ページは学習・参照用の整理資料です。**最終判断は原本PDF・関係部門の指示に従ってください。**
- 社外公開時は、顧客名、内部システム名、文書番号、個人名などの機密・特定情報を掲載しないでください。

- 본 페이지는 학습/참고용 정리본입니다. **최종 판단은 원문 PDF 및 관계 부서 지시에 따르세요.**
- 외부 공개 시 고객명/내부시스템명/문서번호/개인명 등 특정 가능한 정보는 기재하지 마세요.

## 使い方 / 사용 방법
- 右上のボタンで **日本語 / 한국어** を切替できます。
- 各ステップ内で **規定 / 営業視点** を切替できます。
- 検索欄でキーワード検索が可能です（例：特採、サンプル、β、DR2）。

- 우측 상단 버튼으로 **일본어/한국어** 전환
- 각 단계 카드에서 **규정/영업 시점** 탭 전환
- 검색창으로 키워드 검색 가능(예: 특채, 샘플, β, DR2)

## 編集方法 / 수정 방법
- `script.js` の `STEPS` データを編集すると、ページ内容が更新됩니다。
- 문구/체크리스트/게이트 조건은 `script.js`의 `STEPS`에서 수정하면 페이지에 반영됩니다.
